# 仿网易云音乐播放器
### 功能介绍
1. 扫描并显示本地歌曲
2. 具备常规播放功能，顺序播放，随机播放，单曲循环
3. 推送排行榜音乐功能
4. 搜索在线音乐功能
5. 下载在线音乐功能
6. 歌词同步滚动功能
7. 锁屏交互界面功能
8. 定时关闭音乐功能
